<objective>
Create the UTXO domain package with persistent UTXO set, undo-log for reversibility, and integrate typed transactions into the existing Block and Chain aggregates with atomic multi-bucket bbolt writes.

Purpose: The UTXO set is how the blockchain tracks who owns what. The undo-log enables future chain reorganization (Phase 4). Atomic writes ensure crash-safe consistency between block storage and UTXO state.
Output: UTXO domain package, bbolt persistence with utxo/undo buckets, modified Block entity with typed transactions, modified Chain aggregate with coinbase and UTXO awareness, updated storage models.
</objective>
