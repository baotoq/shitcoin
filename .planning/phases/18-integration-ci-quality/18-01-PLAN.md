  <name>Task 1: P2P multi-node integration tests</name>
  <files>internal/integration/integration_test.go</files>
  <action>
Create `internal/integration/integration_test.go` (package `integration_test`) with the following tests:

**Helper function `setupNode`:** Creates a fully wired node (mockChainRepo, mockUTXORepo, utxo.Set, chain.Chain, mempool, p2p.Server) with given ChainConfig and miner address. Uses port 0 for OS-assigned ports. Registers t.Cleanup for server Stop. Returns the server, chain, mempool, and utxoSet.

**TestIntegration_TwoNodeHandshake:**
- Create two nodes with same ChainConfig (InitialDifficulty: 1, GenesisMessage: "integration-test", BlockReward: 5000000000) and same miner address.
- Connect node B to node A via `srvB.Connect(srvA.ListenAddr())`.
- Use `require.Eventually` to assert both nodes have PeerCount() == 1 (5s timeout, 50ms poll).

**TestIntegration_BlockSync:**
- Create node A, initialize, mine 2 additional blocks (total height 2).
- Create node B with same config/miner, initialize (height 0).
- Connect B to A.
- Use `require.Eventually` to assert `chainB.Height() == chainA.Height()` (5s timeout, 50ms poll).
- Verify node B's chain height equals node A's chain height.

**TestIntegration_TxRelay:**
- Create two nodes with same config, connect them.
- Wait for peer connection to establish.
- Create a wallet, get UTXOs from node A's utxoSet (from genesis coinbase).
- Build a signed transaction using `testutil.MustBuildSignedTx`.
- Add tx to node A's mempool via `poolA.Add(signedTx)`.
- Broadcast tx from node A via `srvA.BroadcastTx(signedTx, "")`.
- Use `require.Eventually` to assert node B's mempool contains the transaction (5s timeout, 50ms poll). Check `poolB.Count() == 1`.

Use `require.Eventually` for ALL async assertions (project convention). Never use `time.Sleep`.

  <action>
Create `internal/integration/e2e_chain_test.go` (package `integration_test`) with the following tests:

**TestE2E_WalletToBalance:**
1. Create sender and receiver wallets via `testutil.MustCreateWallet(t)`.
2. Setup chain with mock repos: `testutil.NewMockChainRepo()`, `testutil.NewMockUTXORepo()`, `utxo.NewSet(utxoRepo)`, `chain.NewChain(repo, &block.ProofOfWork{}, cfg, utxoSet)` with InitialDifficulty: 1, BlockReward: 5000000000.
3. Initialize chain with sender's address -- this creates genesis block with coinbase to sender.
4. Verify sender has exactly 1 UTXO via `utxoSet.GetByAddress(sender.Address())`.
5. Build a signed transaction sending from sender using `testutil.MustBuildSignedTx(t, utxoSet, sender.PrivateKey(), sender.Address())`. Note: MustBuildSignedTx creates a tx spending from sender's UTXOs. If the builder does not support specifying a receiver, the test should still verify the UTXO state change after mining.
6. Mine a block containing the signed transaction: `ch.MineBlock(ctx, sender.Address(), []*tx.Transaction{signedTx}, 0)`.
7. Verify chain height is now 1 (genesis=0, mined=1).
8. Verify sender's UTXOs have changed (different count or amounts than before the tx).

**TestE2E_MineMultipleBlocks:**
1. Create a wallet, setup chain with mock repos (same config as above).
2. Initialize chain.
3. Mine 5 blocks in a loop with no transactions.
4. Verify `ch.Height() == 5`.
5. Verify the miner's UTXOs -- should have 6 UTXOs (1 genesis coinbase + 5 block coinbases).

**TestE2E_MempoolIntegration:**
1. Create wallet, setup chain + utxoSet + mempool.
2. Initialize chain (gives wallet a coinbase UTXO).
3. Build a signed tx using testutil helper.
4. Add tx to mempool via `pool.Add(signedTx)` -- verify no error and `pool.Count() == 1`.
5. Get pending txs from mempool via `pool.GetAll()`.
6. Mine block with pending txs: `ch.MineBlock(ctx, addr, pendingTxs, 0)`.
7. Verify mempool is empty after mining (pool.Count() == 0 -- if mempool auto-clears, or verify tx is in block).
8. Verify UTXO state reflects the mined transaction.
