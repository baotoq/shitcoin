  <behavior>
    - AdjustDifficulty(currentBits, actualTimeSpan, targetTimeSpan): ratio < 1 (fast blocks) increases bits, ratio > 1 (slow blocks) decreases bits
    - AdjustDifficulty clamps adjustment factor to [0.25, 4.0] range
    - AdjustDifficulty clamps result bits to [1, 255] range
    - Repository.SaveBlock persists a block to bbolt and it can be retrieved by hash
    - Repository.GetBlockByHeight returns the correct block for a given height
    - Repository.GetLatestBlock returns the most recently saved block
    - Repository.GetChainHeight returns the current chain height
    - Blocks survive bbolt close/reopen (persistence across restarts)
    - Chain.Initialize creates and persists genesis block if chain is empty
    - Chain.Initialize loads existing chain if blocks exist in storage
    - Chain.MineBlock creates a new block referencing the latest block, mines it with PoW, persists it
    - Chain.MineBlock adjusts difficulty every DifficultyAdjustInterval blocks
  </behavior>
  <action>
    1. Create `internal/domain/block/difficulty.go`:
       - AdjustDifficulty(currentBits uint32, actualTimeSpan, targetTimeSpan time.Duration) uint32
       - Calculate ratio = actual / target
       - Clamp ratio to [0.25, 4.0]
       - newBits = float64(currentBits) / ratio (faster blocks = higher bits = harder)
       - Clamp result to [1, 255]
       - Return uint32(math.Round(newBits))

    2. Create `internal/domain/block/difficulty_test.go` (write tests FIRST):
       - TestAdjustDifficulty_BlocksTooFast: actual=5s, target=10s -> bits increase (harder)
       - TestAdjustDifficulty_BlocksTooSlow: actual=20s, target=10s -> bits decrease (easier)
       - TestAdjustDifficulty_OnTarget: actual=10s, target=10s -> bits unchanged
       - TestAdjustDifficulty_ClampMax: actual=0.1s, target=10s -> clamp at 4x increase
       - TestAdjustDifficulty_ClampMin: actual=100s, target=10s -> clamp at 0.25x decrease
       - TestAdjustDifficulty_ClampBitsRange: edge cases for bits near 1 and 255

    3. Create `internal/domain/chain/repository.go`:
       - Repository interface with methods:
         - SaveBlock(ctx context.Context, b *block.Block) error
         - GetBlock(ctx context.Context, hash block.Hash) (*block.Block, error)
         - GetBlockByHeight(ctx context.Context, height uint64) (*block.Block, error)
         - GetLatestBlock(ctx context.Context) (*block.Block, error)
         - GetChainHeight(ctx context.Context) (uint64, error)
         - GetBlocksInRange(ctx context.Context, startHeight, endHeight uint64) ([]*block.Block, error)

    4. Create `internal/domain/chain/errors.go`:
       - var ErrBlockNotFound, ErrChainEmpty, ErrInvalidPrevHash

    5. Create `internal/domain/chain/chain.go`:
       - Chain struct with: repo (Repository), pow (*block.ProofOfWork), latestBlock (*block.Block), consensusConfig (holds block time target, difficulty interval, initial difficulty, genesis message)
       - NewChain(repo Repository, pow *block.ProofOfWork, config ChainConfig) *Chain
       - ChainConfig struct: BlockTimeTarget int, DifficultyAdjustInterval int, InitialDifficulty int, GenesisMessage string
       - Initialize(ctx context.Context) error: check if chain exists (GetLatestBlock). If ErrChainEmpty, create genesis block with config.GenesisMessage and config.InitialDifficulty bits, mine it, save it. If chain exists, load latest block.
       - MineBlock(ctx context.Context) (*block.Block, error): create new block referencing latest block's hash, determine bits (check if difficulty adjustment is due -- every DifficultyAdjustInterval blocks, call AdjustDifficulty using timestamps of blocks in the adjustment window), mine with PoW, save, update latestBlock.
       - LatestBlock() *block.Block, Height() uint64 getters
       - getCurrentBits(ctx) helper: if height+1 is a multiple of DifficultyAdjustInterval, compute new bits using AdjustDifficulty. Otherwise return latest block's bits.

    6. Create `internal/infrastructure/persistence/bbolt/storage_model.go`:
       - BlockModel and HeaderModel structs with EXPORTED fields and json tags for JSON serialization
       - BlockModelFromDomain(b *block.Block) *BlockModel -- converts domain block to storage model
       - (bm *BlockModel) ToDomain() *block.Block -- converts storage model back to domain using ReconstructBlock
       - Store timestamp as int64 Unix seconds, hash as hex string, for JSON debuggability

    7. Create `internal/infrastructure/persistence/bbolt/chain_repo.go`:
       - BboltRepository struct holding *bolt.DB
       - NewBboltRepository(db *bolt.DB) (*BboltRepository, error): creates buckets ("blocks", "chain_meta") if not exist
       - Implement all chain.Repository methods
       - Buckets: "blocks" stores block JSON by hash key AND height index (h:<8-byte-big-endian>), "chain_meta" stores "latest_hash" and "height"
       - CRITICAL: Copy byte slices inside bolt transaction callbacks before they close (bbolt pitfall #4)
       - SaveBlock: single bolt.Update transaction that stores block JSON, height index, and updates chain_meta
       - GetBlock: bolt.View, lookup by hash, copy data, unmarshal, convert to domain
       - GetBlockByHeight: bolt.View, lookup height index to get hash, then get block by hash
       - GetLatestBlock: read latest_hash from chain_meta, then GetBlock. Return chain.ErrChainEmpty if no latest_hash.
       - GetChainHeight: read height from chain_meta. Return 0 if not set.
       - GetBlocksInRange: iterate height index from startHeight to endHeight

    8. Create `internal/infrastructure/persistence/bbolt/chain_repo_test.go` (write tests FIRST):
       - Use t.TempDir() for bbolt database file
       - TestSaveAndGetBlock: save a block, retrieve by hash, verify all fields match
       - TestGetBlockByHeight: save 3 blocks, retrieve each by height
       - TestGetLatestBlock: save 3 blocks, latest is the last one saved
       - TestGetChainHeight: empty=0, after 3 blocks=2 (0-indexed height)
       - TestChainPersistence: save blocks, close DB, reopen, verify blocks still retrievable
       - TestGetBlockNotFound: get non-existent hash returns ErrBlockNotFound
       - TestEmptyChainLatestBlock: returns ErrChainEmpty

    9. Run all tests: `go test -race ./... -count=1`
