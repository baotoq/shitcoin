<task type="auto" tdd="true">
  <name>Task 2: ProofOfWork mining service with validation</name>
  <files>
    internal/domain/block/pow.go,
    internal/domain/block/pow_test.go
  </files>
  <behavior>
    - BitsToTarget(bits) converts target bits to big.Int target (1 << (256 - bits))
    - Mine(block) finds a nonce where block hash < target, sets nonce and hash on the block
    - Mine returns ErrNonceExhausted if no valid nonce found (test with impossibly high bits like 256)
    - Validate(block) returns true if block hash < target for block's bits
    - Validate(block) returns false if block hash >= target (tampered nonce)
    - Mining a genesis block with InitialDifficulty=16 bits succeeds in reasonable time
    - Mined block's hash starts with expected leading zero bits
  </behavior>
  <action>
    1. Create `internal/domain/block/pow.go`:
       - ProofOfWork struct (stateless domain service)
       - BitsToTarget(bits uint32) *big.Int: creates target = 1 << (256 - bits). Use math/big.
       - Mine(b *Block) error: loop nonce from 0 to math.MaxUint32, for each: set nonce on header, compute header hash, convert to big.Int, compare to target. If hash < target, set block hash and return nil. If exhausted, return ErrNonceExhausted.
       - Validate(b *Block) bool: recompute header hash, compare to target from block's bits. Return true if hash < target.

    2. Create `internal/domain/block/pow_test.go` -- write tests FIRST, then implement:
       - TestBitsToTarget: verify target for bits=16 equals 2^240, target for bits=1 equals 2^255
       - TestMineGenesisBlock: create genesis block with bits=16, mine it, verify hash is non-zero and hash < target
       - TestMineValidation: mine a block, validate returns true; tamper nonce, validate returns false
       - TestMineDeterministic: same block header with same nonce produces same hash
       - TestMineNonceExhausted: create block with bits=256 (impossible target), expect ErrNonceExhausted
       - TestMineMultipleBlocks: mine 3 blocks in sequence (each referencing previous hash), all validate

    3. Run all domain tests to ensure nothing is broken: `go test ./internal/domain/block/... -v -count=1`
